Demo Hệ Thống Chatbox

Hệ thống chat trực tuyến với WebSocket và hỗ trợ admin

Tính Năng

  • Chat thời gian thực với WebSocket
  • Giao diện responsive, tương thích mobile
  • Hỗ trợ chat khách vãng lai (không cần đăng ký)
  • Lưu trữ lịch sử chat trong database
  • Thông báo tin nhắn chưa đọc cho admin
  • Giao diện quản lý chat cho admin
  • Fallback REST API khi WebSocket không khả dụng

Công Nghệ

  • Spring Boot + WebSocket
  • JPA/Hibernate + H2 Database
  • JavaScript ES6 + SockJS + Stomp
  • Bootstrap 5 + Font Awesome
  • Spring Security
  • Responsive Design

Hướng Dẫn Sử Dụng

Cho Khách Hàng:
  1. Nhấn vào nút chat ở góc dưới bên phải
  2. Nhập tên và email (tùy chọn)
  3. Gửi tin nhắn và chờ admin phản hồi
  4. Chat được lưu theo session ID
Cho Admin:
  1. Truy cập trang quản lý chat
  2. Xem danh sách phiên chat và tin nhắn chưa đọc
  3. Chọn phiên chat để xem và trả lời
  4. Tin nhắn được đồng bộ real-time
